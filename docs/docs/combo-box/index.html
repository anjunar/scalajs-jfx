Forms
jfx.form.ComboBox
ComboBox
A customizable selection control with renderable values and dropdown rows. The combo box API supports custom item rendering, custom selected-value rendering and overlay-based dropdown behaviour.
Operational Fit
Can be styled as a product-grade picker instead of a default select.
Selected value and row rendering are both customizable.
Plays well with the viewport and overlay system.
API Snapshot
comboBox[T](name) { ... }
Create a typed selection field.
items
Bind selectable items via ListProperty.
itemRenderer
Customize dropdown row content.
valueRenderer
Customize the selected value presentation.
Import Surface
import jfx.core.state.ListProperty import jfx.form.ComboBox.{comboBox, items, itemRenderer, valueRenderer}
Usage Skeleton
val teams = ListProperty(js.Array("Platform Engineering", "Design Systems", "Field Research")) comboBox[String]("team") { items = teams dropdownHeightPx = 272.0 rowHeightPx = 68.0 itemRenderer = { ...custom row layout... } valueRenderer = { ...selected value layout... } }
Integration Patterns
Custom Brand Feel
Render both the selected value and the dropdown rows so the control matches the rest of the product.
Overlay Based
Use it when the picker should float cleanly over nearby layout instead of expanding inline.
Typed Selection
Keep the selected model type explicit so form binding remains predictable.
Live Sandbox
The docs example uses custom row and selected-value rendering so the combo box feels like a product component instead of a default select.
