Logic
jfx.statement.Conditional
Conditional (when)
Reactive branching in the DSL based on a boolean property. Use when you need to show or hide parts of the UI based on state changes.
Operational Fit
Supports 'thenDo' and 'elseDo' branches.
Efficiently mounts and unmounts DOM nodes as the condition changes.
Integrates with the DSL scope and form system.
API Snapshot
conditional(property) { ... }
Start a conditional block.
thenDo { ... }
Define the content when the property is true.
elseDo { ... }
Define the content when the property is false.
Import Surface
import jfx.statement.Conditional.{conditional, thenDo, elseDo}
Usage Skeleton
val showDetails = Property(false) conditional(showDetails) { thenDo { div { textContent = "Detailed information here..." } } elseDo { div { textContent = "Click to show details." } } }
Integration Patterns
Feature Toggles
Use conditional to hide features that are not yet enabled for the user.
Loading States
Show a spinner while data is loading and the content once it's ready.
Empty States
Display an empty state message when a list has no items.
