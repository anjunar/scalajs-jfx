jfx.form.ImageCropper
ImageCropper
Media editing inside a typed form workflow. The image cropper is meant for workflows where choosing and shaping media should stay inside the same model-editing surface.
Operational Fit
Works on Media-backed model properties.
Supports window-based editing flows.
Pairs naturally with thumbnail generation and preview UIs.
API Snapshot
imageCropper(name) { ... }
Bind the cropper to a media property.
aspectRatio
Control the crop frame ratio.
outputMaxWidth / outputMaxHeight
Limit generated output size.
windowTitle
Label the cropper editing window.
Import Surface
import jfx.form.ImageCropper.imageCropper
Usage Skeleton
inputContainer("Avatar") { imageCropper("media") { placeholder = "Noch kein Bild ausgewählt" windowTitle = "Bild zuschneiden" aspectRatio = 1.0 outputMaxWidth = 512 outputMaxHeight = 512 } }
Integration Patterns
Profile And CMS Flows
Use it when upload, crop and preview should all stay inside one form experience.
Window Editing
Launch the crop step in a focused window so the main form remains uncluttered.
Controlled Output
Limit output size up front so downstream thumbnails and payload sizes stay predictable.
