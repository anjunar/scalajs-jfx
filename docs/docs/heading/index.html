jfx.control.Heading
Heading
Semantic text headings from H1 to H6. Headings help structure your content and provide visual hierarchy to your pages.
Operational Fit
Supports levels 1 through 6.
Normalizes out-of-bounds levels to valid HTML heading tags.
Promotes semantic HTML and better accessibility.
API Snapshot
heading(level) { ... }
Create a heading of a specific level (1-6).
headingLevel
Access the normalized level of the heading.
Import Surface
import jfx.control.Heading.heading
Usage Skeleton
heading(2) { textContent = "Component Reference" }
Integration Patterns
Page Titles

Typography
Apply custom styles to headings to match your brand's visual language.
