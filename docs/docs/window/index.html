Layout
jfx.layout.Window
Window
Floating, draggable windows for secondary workflows. Windows provide a desktop-like experience within the browser, allowing for multi-tasking and non-blocking workflows.
Operational Fit
Draggable and resizable by the user.
Supports title bars with status indicators and close buttons.
Managed by the Viewport for proper layering and focus handling.
API Snapshot
Viewport.addWindow(conf)
Launch a new window in the shared viewport.
WindowConf
Configure title, initial size and content.
Viewport.captureComponent { ... }
Wrap DSL content for window usage.
Import Surface
import jfx.layout.Viewport.WindowConf import jfx.layout.Viewport
Usage Skeleton
Viewport.addWindow( WindowConf( title = "Edit Profile", width = 400, height = 300, component = Viewport.captureComponent { vbox { textContent = "Profile Editor Content" } } ) )
