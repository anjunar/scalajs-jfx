Control
jfx.control.Image
Image
Display images with reactive source and styling. Image provides a way to render images while observing source changes and providing easy styling hooks.
Operational Fit
Supports reactive source (URL) property.
Easy to style with CSS for size, object-fit and borders.
Standard HTML img element under the hood for maximum compatibility.
API Snapshot
image(srcProperty) { ... }
Create an image bound to a property.
srcProperty
The underlying property holding the image URL.
Import Surface
import jfx.control.Image.image
Usage Skeleton
val avatarUrl = Property("https://example.com/avatar.png") image(avatarUrl) { style = "width: 48px; height: 48px; border-radius: 50%; object-fit: cover;" }
