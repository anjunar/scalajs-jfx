Action
jfx.action.Button
Button
A standard clickable button for triggering actions and form submissions. Buttons are the primary way for users to interact with your application logic or submit forms.
Operational Fit
Automatically switches type to 'submit' inside forms.
Supports custom click listeners via the onClick DSL.
Easily styled with custom CSS classes.
API Snapshot
button(label) { ... }
Create a button with a text label.
onClick { event => ... }
Register a click listener.
buttonType
Get or set the underlying HTML button type.
disabled
Control the button's interactive state.
Import Surface
import jfx.action.Button.{button, onClick}
Usage Skeleton
button("Save Changes") { onClick { _ => println("Saving data...") } }
Integration Patterns
Form Submission
Buttons inside a form default to 'submit' type, triggering the form's onSubmit handler.
Icon Buttons
Combine buttons with icon components or custom CSS for compact action triggers.
Scoped Actions
Use buttons to trigger scope-aware actions like opening windows or navigating.
